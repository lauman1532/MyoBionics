Getting The Materials

The following lists out the necessary equipment and materials required to use MyoBionics

  • Thalmic Labs Myo™ Armband
  • Open Bionics™ Ada Hand Kit
  • CC2540/1 Bluetooth Low Energy Module (HM-10/11 Chip) - Example
  • CC Debugger/Alternative solution
  • Soldering Kit, Wires and Solder
  • Computer with Arduino IDE Installed

Preparing the Materials/Resources

1) Print up the Hand

Firstly, as soon as possible, print out the Ada Hand following this guide. *IMPORTANT* Do not assemble the hand yet as some modifications need to be made before assembly.


2) Setting up the Bluetooth Module

The next thing to do is to Setup MyoBridge ,an open-sourced firmware for CC2540/1 BLE module and an Arduino library. This includes the flashing of the BLE Module (typically HM-10/11) and downloading the respective library file. Now the BLE Module is ready to interact with the Myo Armband. The following connections should be available to connect to the pins of the microcontroller used.

  • RX
  • TX
  • Vcc
  • Ground
  • (Optional) Reset
These connections can be connected to a 4-pole 3.5mm Jack for easier interface with the Almond PCB. Follow this guide to use this method


3) Testing the Motors and Almond PCB

Not forgetting the Ada Hand Kit, it is important to check if it works. To do this, download FingerLib.h, a Arduino library by vroland required to operate the Almond PCB. Install the library the same way you installed the MyoBridge library. Next attach the motors onto the Almond PCB:

  1. Place the Almond Board upside down (Power port facing down)
  2. Gently unclip the motor connectors.
  3. Then slide the finger motors into the port with the copper pads facing into the board.
  4. Slide back the connector clips to secure the motor connectors.
If you are still unsure, you can follow Section 4 of this guide.

Next, attach the power supply (6V-12V; Recommended:12V) to the Almond PCB. This should cause some LEDs on the Almond to light up. If no LEDs light up, the best solution is to contact Open Bionics. Now connect the USB port on the Almond using a USB to USB Micro2.0 cable to a computer with Arduino IDE software installed. Look in the examples folder of the FingerLib library file for "MotorTest". Open the file "MotorTest.ino" using the Arduino IDE and upload the code*.

*IMPORTANT Before uploading, check the settings under the "Tools" tab that the board selected is "Arduino/Genuino Mega 2560". Also ensure that the COM port selected is the correct port

Once uploaded, the motors will extend to the maximum. This shows that the code is correctly uploaded. Open the Serial Monitor and set the baud rate to 38400 and you should see m0-m4 printed out with some numbers. By sending the corresponding finger number in the Serial Monitor, you can control the extension/contraction of the finger. This just checks if the Almond and motors are working as intended.

This is also covered in Section 6 of the Ada V1.1 Assembly Guide themselves.

4) Attaching the Bluetooth Module to the Almond PCB

Foremost, if you have attached the Bluetooth Module to the 4-port 3.5mm Jack, just plug the jack into the Almond. Don't forget to short the resistor pads for UART Serial communication.

If you are not using the Audio Jack, you will need to solder the wires from the Bluetooth Module directly onto the Almond. Follow the image attached in this section for a complete guide.

NOTE: If the RX and TX pins are swapped, don't worry, a minor change in the code will fix it.

Once you have attached the Bluetooth Module to the Almond PCB, we need to test the connection. To do this, use the code "printFirmwareInfo.ino" in the examples folder of MyoBridge library. Alter the line SoftwareSerial bridgeSerial(2,3); to SoftwareSerial bridgeSerial(51,52); . Once this is done, upload the code* and open the Serial Monitor and set the baud rate to 115200. Make sure that the Myo Armband is on and is not connected to anything. It should then try to connect to the Bluetooth Module and the connection status is printed on the Serial Monitor. If the line "connected!" is printed on the Serial Monitor, then that means that the Myo Armband can succesfully connect to the Bluetooth Module and Almond PCB.

Assembly Time

Now that everything is prepared and all tests have been done, you can completely assemble the hand. For further simplicity, follow the Ada V1.1 Assembly Guide made by Open Bionics themselves. It is recommended that you attach everything accordingly but do not attach the back cover of the Ada Hand.

Once this is done, you should have the Almond mounted on the Ada palm attached to the fingers. Depending on which Bluetooth Module is used and how it is attached, it may or may not fit in the tray itself. Should it fit, the Bluetooth Module can be fitted closely to the Almond within the tray and covered with the back cover. However, if it does not fit, a solution will be to have the Bluetooth Module attached on the back cover. This can be done by applying glue to the flat back of the Bluetooth Module and attaching it to the back cover.

Now that the assembly of the hand is complete, download the MyoBionics library file and install it. Select the appropriate folder to your setup and upload the code and you're done! If there are any issues, contact us or check out our Troubleshooting section.
